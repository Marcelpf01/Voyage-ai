# 🌍 Overview

Voyage AI is a modern full-stack travel planning platform powered by Google Gemini AI.

Instead of spending hours researching flights, attractions, restaurants, transportation, and budgeting, Voyage AI creates intelligent personalized itineraries tailored to each traveler.

The project combines modern frontend engineering with AI integration, authentication, cloud databases, responsive design, and interactive maps into a production-style SaaS application.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Marcelpf01/voyage-ai.git
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

---

# 🔑 Environment Variables

Create a `.env.local`

```env
NEXT_PUBLIC_FIREBASE_API_KEY=

NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=

NEXT_PUBLIC_FIREBASE_PROJECT_ID=

NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=

NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=

NEXT_PUBLIC_FIREBASE_APP_ID=

NEXT_PUBLIC_GEMINI_API_KEY=

NEXT_PUBLIC_GOOGLE_MAPS_API_KEY=
```
